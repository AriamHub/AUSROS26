# Australian School of Robotic Systems AUSROS 2026

AUSROS 2026 is a week-long summer school covering the foundations of modern
robotics: sensing, estimation, control, planning, learning, and SLAM: through
core lectures, Deep Dive talks, hands-on sessions, and workshops.
